Compilando a DRV Game Engine

Esse manual explica como compilar a DRV Engine nas diversas plataformas suportadas, assim como alguns conceitos de como funciona o sistema de compilação escolhido.

VS Code

A ferramenta de desenvolvimento padrão escolhido para o desenvolvimento é o VS Code. Você pode baixa-lo e instala-lo a partir de https://code.visualstudio.com/download. Para usa-lo é muito simples. Basta baixar a extenção c++ para VS Code. O arquivo c_cpp_properties.json já está configurado corretamente para o projeto. Apenas selecione no canto inferior direito qual das configurações deseja usar. Por exemplo, caso deseje trabalhar com Windows no modo debug, selecione "Windows (Debug)" para que o intellisense possa mostrar a sintaxy correta, como pode ser visto na imagem. É recomendado que instale todas as extenções sugeridas em "extensions.json" para melhor desenvolvimento. Ao iniciar o VS Code pela primeira vez, o mesmo ira recomendar que instale as extenções contidas em "extensions.json".

Observação: caso esteja com um arquivo .md aberto, a opção de cofirugração não irá aparecer no canto inferior direito. Apenas abra um arquivo .c, .h,.cpp ou .hpp para que elas sejam exibidas em seu VS Code.
